⚡ Calculadora de Consumo de Energia
📌 Sobre o Projeto

Este projeto foi desenvolvido para calcular o consumo mensal de energia elétrica de aparelhos domésticos de forma simples e rápida.

O usuário informa:

Nome do aparelho
Potência (em watts)
Horas de uso diário

O sistema calcula automaticamente o consumo mensal em kWh e o custo estimado da energia elétrica.

🚀 Tecnologias Utilizadas






🧮 Fórmula Utilizada

Consumo mensal (kWh):

consumo = (potência × horas por dia × 30) / 1000

Custo estimado:

custo = consumo × valor do kWh
▶️ Como Executar
Clone o repositório:
git clone https://github.com/seu-usuario/consumo-energia.git
Acesse a pasta:
cd consumo-energia
Execute o programa:
python app.py
💡 Exemplo de Uso
Digite o nome do aparelho: Televisão
Digite a potência (em watts): 30
Digite as horas de uso por dia: 10

--- Resultado ---
Aparelho: Televisão
Consumo estimado: 9.00 kWh/mês
Custo estimado: R$ 6.75
